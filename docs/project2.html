Project 2 - Sales Insights - Pivot Tables in Excel

Overview:

STEPS:
STEP 1: Created 3 new sheets:
a. Original Data – to make sure if you mess something up you still have a sheet with the correct data.
b. Working sheet – A sheet you have to edit the data with
c. Pivot Table sheet – A sheet where you create all the pivot tables you may need
d. Dashboard sheet.

STEP 2: Data Cleaning and Transformation
a. Removed all duplicates using the "Remove Duplicates" feature in Excel.
b. Improved data clarity by updating the Marital Status column, changing 'M' to "Married" and 'S' to "Single." Similarly, the Gender column was updated by changing 'M' to "Male" and 'F' to "Female." This ensures the data is as clear as possible for analysis, reducing the risk of confusion.
c. Used nested IF statements to categorize the age data into groups: Adolescent, Middle-aged, and Old. These age categories enhance the usability of the data in the dashboard, compared to listing individual ages from 15 to 71.

STEP 3: Created 3 Pivot Charts on a separate sheet.
a. Average Income Per Purchase
b. Customer Age bracket Per Purchase
c. Customer Commute Per Purchase

STEP 4: Created slicers to allow the user to filter the data!
a. Added the slicer for Marital Status
b. Also Added Region
c. Also Added Education

STEP 5: The last functionality added to the Dashboard was an ID search: This allows the user to type in any User ID and the search will auto populate with all of the data related to that ID.
